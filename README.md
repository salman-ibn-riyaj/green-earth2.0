# 🌿 greenEarth - Plant E-Commerce Platform

A modern, fast, and responsive frontend application built with **Next.js** for a plant e-commerce platform. This user-friendly interface allows plant lovers to seamlessly browse trees by category, view detailed descriptions, manage their shopping cart, and interact with a secure account authentication system.

## 🔗 Live Link
Experience the live application here: [https://green-earth-tau.vercel.app](https://green-earth-tau.vercel.app)

## 🚀 Key Features

* **User Authentication UI:** Clean and responsive forms for user registration (Sign-up) and login.
* **Dynamic Category Browsing:** Efficient product layout that allows users to explore trees and plants by specific categories.
* **Detailed Product View:** Dedicated product pages to display plant characteristics, pricing, and description.
* **Interactive Shopping Cart:** Real-time cart management system where users can easily add items, adjust quantities, or remove products.
* **Fully Responsive Design:** Fully optimized with modern UI principles to look beautiful on mobile, tablet, and desktop screens.

---

## 🛠️ Tech Stack

* **Framework:** Next.js (React)
* **Styling:** Tailwind CSS / CSS
* **State Management:** Context API / React Hooks (for handling cart state and user session UI)

## ⚙️ How to Run Locally

Follow these quick steps to get the project running on your local machine:

### 1. Clone the Repository
```bash
git clone [https://github.com/salman-ibn-riyaj/greenEarth.git](https://github.com/salman-ibn-riyaj/greenEarth.git)
cd greenEarth
npm install
npm run dev
Now, open http://localhost:3000 with your browser to see the live local version.
